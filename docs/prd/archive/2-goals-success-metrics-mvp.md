# 2. Goals & Success Metrics (MVP)
**Goals**
- Replace Zoho People for EOR timesheets, leave, and payslip/document access by MVP.
- Provide self‑service profile updates for 100% of active EORs.
- Enforce one active client assignment per EOR with clean transition flows and audit history.
- Support multi‑country operations at launch (India, Sri Lanka, Philippines).

**Success Metrics**
- **Activation:** ≥ 90% of invited EORs complete onboarding within 14 days.
- **Engagement:** ≥ 95% weekly timesheet submission; ≥ 90% leave requests via portal.
- **Operational/Cost:** ≥ 30% reduction in HR SaaS spend vs Zoho baseline by month 3 post‑launch; < 5 minutes median Ops invite‑to‑onboard.
- **Quality/Uptime:** < 1% data error across core fields post‑migration; 99.9% monthly uptime.
