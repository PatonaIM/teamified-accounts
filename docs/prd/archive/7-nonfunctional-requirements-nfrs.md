# 7. Non‑Functional Requirements (NFRs)
- **Security/Privacy:** PII encryption in transit/at rest; RBAC; audit logging for sensitive events; least‑privilege defaults.
- **Availability:** 99.9% monthly uptime; graceful degradation for documents.
- **Performance:** P95 page loads < 2.0s on baseline broadband; file upload ≤ 10MB within 10s.
- **Accessibility:** WCAG 2.1 AA considerations; keyboard navigation; focus states.
- **Compatibility:** Latest Chrome/Edge/Safari/Firefox; responsive for mobile.
- **Observability:** Structured logs, error tracking, basic dashboards for traffic and errors.
